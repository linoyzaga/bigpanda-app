# Big Panda Application
 
In the project directory, you need to run:

#### `yarn install`

Install app dependencies.<br>

#### `yarn dev`

Runs the app.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.<br>
The server runs on port 3001.
